# PhasorReleases
## What is Phasor
Phasor is an EDA tool built for collaboration between agents and users (you!) to find out more, visit [our website](https://phasoreda.com), where you can find a link to join our discord
## What is this repo?
Phasor is closed source for now, so this repo is just for releases. We don't recommend you download from here though -- [our download page](https://phasoreda.com/download) makes that a lot easier.
